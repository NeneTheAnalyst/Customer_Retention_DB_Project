# Customer Retention and Loyalty Analytics Database

This repository contains SQL scripts and related files for a **Customer Retention and Loyalty Analytics** project. 
The database is designed for a transaction-based retail system and supports advanced analytics for customer behavior, loyalty program performance, and marketing effectiveness.

## Contents

- **SQL Scripts**: Scripts to create tables, indexes, stored procedures, and user-defined functions.  
- **ERD Diagram**: Entity-Relationship Diagram showing tables, primary/foreign keys, and relationships.  
- **README.md**: Project description and usage instructions.

## Features

- **Normalized Database**: Achieves Third Normal Form (3NF) to reduce redundancy and maintain data integrity.  
- **Loyalty Analytics**: Track customer engagement, loyalty program participation, and calculate Customer Lifetime Value (CLV).  
- **Marketing Analytics**: Analyze the effectiveness of email and social media campaigns.  
- **Advanced SQL**: Uses joins, CTEs, window functions, aggregation, and stored procedures for insights.  
- **Optimized Queries**: Includes indexes and query rewriting for better performance on large datasets.

## Usage

1. Create the database in MySQL.  
2. Run the provided SQL scripts in order to build tables, constraints, and populate sample data.  
3. Use the stored procedures and functions to run analytics queries.
